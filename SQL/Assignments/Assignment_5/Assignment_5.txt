Q26) Write an SQL query to display member ID with highest and lowest number of books
issued.
-- (SELECT member_id, COUNT(book_no) AS total_books
    ->  FROM issue
    ->  GROUP BY member_id
    ->  ORDER BY total_books DESC
    ->  LIMIT 1)
    -> UNION
    -> (SELECT member_id, COUNT(book_no) AS total_books
    ->  FROM issue
    ->  GROUP BY member_id
    ->  ORDER BY total_books ASC
    ->  LIMIT 1);
